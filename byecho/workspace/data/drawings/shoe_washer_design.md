**版本**: V1.0

## 一、产品概述

### 1.1 产品定位
面向家庭/商用场景的自动化洗鞋设备，支持自动清洗、烘干、消毒、存储一体化功能。

### 1.2 核心功能
| 功能 | 描述 |
|------|------|
| 自动清洗 | 高压喷淋 + 旋转刷洗，支持多种清洗模式 |
| 热风烘干 | PTC加热 + 循环风道，60℃恒温烘干 |
| UV消毒 | 紫外线杀菌，99.9%灭菌率 |
| 智能存储 | 清洗完成后自动转入存储模式，保持干燥 |
| APP控制 | 远程启动、进度查看、耗材提醒 |

### 1.3 技术规格
| 参数 | 规格 |
|------|------|
| 容量 | 6双鞋 (标准柜) / 12双鞋 (商用柜) |
| 电源 | AC 220V 50Hz |
| 额定功率 | 800W (清洗) / 1200W (烘干) |
| 水箱容量 | 8L 清水箱 + 6L 污水箱 |
| 防护等级 | IPX4 (内部电子仓 IP67) |
| 工作噪音 | ≤55dB |
| 单次清洗时间 | 15-30分钟 (可选模式) |

### 2.4 电气安全设计
- **漏电保护**: 内置30mA漏电保护器
- **过流保护**: 各支路独立保险丝
- **过热保护**: PTC内置温控 + 软件双重保护
- **防水设计**: 电子仓独立密封，IP67防护
- **接地设计**: 金属外壳可靠接地

## 五、风险评估

| 风险项 | 等级 | 应对措施 |
|--------|------|----------|
| 防水设计不达标 | 高 | 早期进行IPX4测试，预留密封设计余量 |
